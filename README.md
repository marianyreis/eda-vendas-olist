# EDA — Vendas e Satisfacao de Clientes (Olist)

Analise exploratoria do dataset publico de e-commerce brasileiro da Olist,
com foco em receita por categoria e satisfacao dos clientes.

## Pergunta central
Quais categorias geram mais receita e os clientes ficam satisfeitos com elas?

## Stack
- Python 3
- pandas
- matplotlib
- seaborn
- Jupyter Notebook

## Estrutura
eda-vendas-olist/
├── data/              # Datasets originais (Olist)
├── notebooks/         # Notebook principal da analise
└── README.md

## Principais resultados
- health_beauty e a categoria com maior receita (R$1,26M) e otima avaliacao (4.14)
- watches_gifts tem o maior ticket medio (R$200) mas volume menor
- bed_bath_table vende muito porem tem a pior avaliacao (3.89)
- Pico de vendas em novembro de 2017 — provavel Black Friday

## Dataset
Olist Brazilian E-Commerce — disponivel no Kaggle
